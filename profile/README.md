## About

L'Odyssée Events is an event company that supports clients from concept to delivery.

We work across event organization, logistics, technical coordination, entertainment, equipment rental and, when useful, digital tools and RFID/NFC solutions such as iFolloww.

## Our Approach

- **Field-first**: tools and processes are designed around real event constraints.
- **Clear interfaces**: teams need fast, understandable workflows during live operations.
- **Reliable deployments**: hardware, software and logistics must hold up on event day.
- **Useful data**: supervision should help decisions while the event is happening.
- **Scalable foundations**: systems should be reusable across different formats and event sizes.
